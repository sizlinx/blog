나혼자 보려고 만든 아치 설치 가이드

부팅 USB 생성

ftp.harukasan.org

에서 iso 파일 다운

하고 Etcher 이나 Rufus 등을 이용해 usb 를 구워준다.

USB 를 UEFI 로 부팅

바이오스에서 부팅 설정 바꾸고 UEFI 로 부팅한다.

왼쪽이 uefi, 오른쪽이 바이오스다.

파티션 설정

lsblk

로 깔고싶은 파티션을 정한다.

cfdisk /dev/[아치리눅스를 깔 파티션]

loop, sdb, sr0 같은건 건드리지 말자.

cfdisk 를 실행하고 나면 맨 위에 GPT 라고 나와있어야한다. 아니면 맨 처음에 물어볼텐데 GPT 하고

primary 어쩌구 나오면 primary 선택하면 된다.

아래 메뉴를 방향키로 움직일 수 있다. New 에서 엔터를 치고 위에서부터 순서대로 1G, 4G, 나머지G (자동) 을 적고 엔터친다.

아래 메뉴에서 Type 으로 위에서부터

EFI System

Linux swap

Linux filesystem

을 선택한다.

DeviceSize Type
/dev/sda11G EFI System
/dev/sda24G Linux swap
/dev/sda3전체 - 5G Linux filesystem

그러면 대충 이런 모양이 나올거다

Write 에 가서 엔터, yes 를 치고 q 로 나온다,

주의: Write 하면 되돌릴 수 없다

포맷

mkfs.vfat -F32 /dev/[EFI 시스템 파티션]
mkfs.ext4 -j /dev/[파일시스템 파티션]
mkswap /dev/[스왑 파티션]
swapon /dev/[스왑 파티션]

마운트

mount /dev/[파일시스템 파티션] /mnt
mkdir /mnt/boot
mount /dev/[EFI 시스템 파티션] /mnt/boot

미러 설정

nano /etc/pacman.d/mirrorlist

Server = 어쩌구 로 시작하는곳 윗줄에

Server = http://ftp.harukasan.org/archlinux/$repo/os/$arch
Server = https://ftp.harukasan.org/archlinux/$repo/os/$arch

를 적는다. ctrl + x, y, 엔터로 빠져나온다.

인터넷 연결

iwctl
station wlan0 scan
station wlan0 connect [연결할 네트워크 이름]
exit

[iwd] 에서 빠져나와서

ping -c 3 8.8.8.8

해서 연결된거같으면 완료

기본 패키지 설치

pacstrap /mnt base base-devel linux linux-firmware nano networkmanager git curl wget vim

따로 다운받고 싶은게 있다면 뒤에 프로그램 이름을 추가하면 된다.

genfstab

genfstab : generate File System table

genfstab -U /mnt >> /mnt/etc/fstab

루트로 아치리눅스 진입

arch-change root

arch-chroot /mnt

locale 설정

vim /etc/locale.gen
# en_US.UTF-8 앞에 주석을 없앤다

echo LANG=en_US.UTF-8 > /etc/locale.conf

시간 설정

시간설정은 설치 다 하고

timedatectl set-ntp true

로 동기화 시킬 수 있으니 패스

기기 시간 설정

hwclock --systohc --utc

로컬 시간 설정

ln -sf /usr/share/zoneinfo/Asia/Seoul /etc/localtime

서울 시간으로 맞춘다

사용자 추가

useradd -m -g users -G wheel -s /bin/bash 유저 이름
EDITOR=vim visudo

하면 나오는 창에서

#%wheel ALL=(ALL) ALL

만 주석을 풀어준다.

비밀번호 설정

passwd [유저이름]
passwd

컴퓨터 이름 설정

echo 컴퓨터이름 > /etc/hostname

부팅설정

bootctl --path=/boot install
vim /boot/loader/loader.conf

vim 창에서는 a 를 눌러서 insert 로 전환해서 다 지우고

default arch
editor 0
timeout 0

이 되게 만든다.

esc/ctrl+[ + :wq 로 저장한다.

vim /boot/loader/entries/arch.conf

를

title ArchLinux
linux /vmlinuz-linux
initrd /initramfs-linux.img

저장하고 빠져나온 후

echo "options root=PARTUUID=$(blkid -s PARTUUID -o value /dev/[파일 시스템 위치]) rw">> /boot/loader/entries/arch.conf

를 하고

cat /boot/loader/entries/arch.conf

로 확인해서

title ArchLinux
linux /vmlinuz-linux
initrd /initramfs-linux.img
options root=PARTUUID=아이디 rw

가 되어있으면 된다.

systemctl enable NetworkManager
exit
umount -R /mnt
reboot

설치 끝
